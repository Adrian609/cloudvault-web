# Setup Instructions

1. Clone repository:
   git clone https://github.com/codewithsach/cloudvault-web.git

2. Go to project folder:
   cd cloudvault-web

3. Create virtual environment:
   python3 -m venv venv
   source venv/bin/activate

4. Install dependencies:
   pip install -r requirements.txt

5. Run application:
   python app.py

6. Open browser:
   http://127.0.0.1:5000
