## How to Capture Screenshots

1. Push a commit or open a pull request so GitHub Actions runs.
2. Open the repository on GitHub.
3. Go to `Actions`.
4. Select the latest `CI/CD` workflow run.
5. Capture the workflow graph after all jobs complete.
6. Open the job summary and capture the telemetry summary.
7. Scroll to `Artifacts` and capture the coverage and security report artifact downloads.
8. Go to `Security -> Code scanning alerts -> CodeQL` and capture the CodeQL results page.
9. Save screenshots into `docs/images/` with these filenames:
   - `workflow-graph.png`
   - `job-summary.png`
   - `codeql-results.png`
   - `coverage-artifact.png`
   - `security-artifacts.png`

These files are referenced by this document and can be used in a portfolio walkthrough.
